# Environment Configuration

## Environment Matrix

| Parameter | Development | Staging | Production |
|---|---|---|---|
| PHP Version | 8.2 | 8.2 | 8.2 |
| Database | SQLite (default) / MySQL 8.0 | MySQL 8.0 | RDS MySQL 8.0 |
| Cache | File | File | TBD (Redis planned) |
| Session | File | File | TBD (Redis planned) |
| Queue | Database | Database | Database (Redis/Horizon planned) |
| Debug | true | false | false |
| APP_ENV | local | staging | production |
| APP_DEBUG | true | false | false |
| HTTPS | Off | On | On |
| Mail | Log | Log | SMTP/SES (configurable) |

## Key Configuration Files

| File | Purpose |
|---|---|
| `.env` | Environment variables (git-ignored) |
| `config/app.php` | Application configuration (name, timezone: Asia/Jakarta, locale: en) |
| `config/auth.php` | Authentication guards, password reset |
| `config/business.php` | Business rules: tax_rate (10%), approval_threshold, bonus_category_id |
| `config/security.php` | Password policy, login throttle, 2FA, backup, audit settings |
| `config/database.php` | Database connections (default: SQLite, MySQL configured) |
| `config/cache.php` | Cache drivers (default: file) |
| `config/session.php` | Session configuration (default: file) |
| `config/queue.php` | Queue connections (default: database, failover) |
| `config/sanctum.php` | API authentication (prefix: apms_, expiry: 480min) |
| `config/services.php` | Third-party: Anthropic AI, Google OAuth, Postmark, Resend, SES |

## Sensitive Values (External to Repo)

- `APP_KEY` — generated by `php artisan key:generate`
- `APP_NAME` — application name
- `DB_*` — database credentials
- `REDIS_*` — Redis credentials (future)
- `MAIL_*` — SMTP credentials
- `AWS_*` — Cloud credentials (future)
- `ANTHROPIC_API_KEY` — AI strategic advisor
- `GOOGLE_CLIENT_ID` / `GOOGLE_CLIENT_SECRET` — Google OAuth
