# Glossary

| Term | Definition |
|---|---|
| **APMS** | Grosir Perfume Management System — the application documented here |
| **Owner** | System administrator with full access, typically the business owner |
| **Admin** | Role with administrative access to manage operations |
| **Cashier** | Role limited to retail POS operations |
| **Wholesale Customer** | B2B customer who purchases via the wholesale portal |
| **Branch (Cabang)** | Physical store location |
| **Grosir** | Indonesian term for wholesale/bulk sales |
| **Eceran** | Indonesian term for retail/individual sales |
| **PPN** | Pajak Pertambahan Nilai (Indonesian VAT, configurable via `config/business.php`, default 10%) |
| **POS** | Point of Sale — retail transaction interface |
| **Invoice** | Transaction receipt number (auto-generated, unique) |
| **Loyalty Tier** | Customer classification level (Bronze/Silver/Gold/Platinum) |
| **Stock Request** | Inter-branch inventory transfer request |
| **Wholesale Order** | B2B order placed via customer portal |
| **Santri** | Islamic boarding school student (used in loyalty/incentive context) |
| **Qris** | Indonesian QR-based payment standard |
| **RajaOngkir** | Indonesian shipping cost/ tracking API |
| **PPOB** | Payment Point Online Bank — bill payment service |
| **Diátaxis** | Documentation framework (Tutorials, How-To, Reference, Explanation) |
| **ADR** | Architecture Decision Record |
| **C4 Model** | Software architecture documentation model (Context, Container, Component, Code) |
| **arc42** | Template for architecture documentation |
| **BABOK** | Business Analysis Body of Knowledge |
