# UI/UX Requirements

## REQ-UI-001: Language

**ID:** REQ-UI-001
**Priority:** High
**Source:** Users

The UI **shall** use Bahasa Indonesia as the primary language. English **may** be supported as a secondary language via lang files.

## REQ-UI-002: Mobile Responsiveness

**ID:** REQ-UI-002
**Priority:** Medium
**Source:** Users

The wholesale customer portal **shall** be fully responsive and usable on mobile devices (320px+ width).

## REQ-UI-003: Dashboard KPIs

**ID:** REQ-UI-003
**Priority:** High
**Source:** Business

The dashboard **shall** display key performance indicators: daily revenue, transaction count, low stock alerts, and pending wholesale orders.

## REQ-UI-004: Search & Filter

**ID:** REQ-UI-004
**Priority:** Medium
**Source:** Usability

Data tables **shall** support search by keyword, filter by status/date, and sortable columns.

## REQ-UI-005: Accessibility

**ID:** REQ-UI-005
**Priority:** Low
**Source:** Standards

The system **should** target WCAG 2.1 Level AA compliance for all customer-facing pages.
