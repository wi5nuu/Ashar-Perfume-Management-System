# Create a Wholesale Customer

1. Navigate to **Wholesale Customers** page
2. Click **Tambah Pelanggan** button
3. Fill in the form:
   - Name, Email, Phone
   - Password (auto-generated)
   - Shipping address
   - Credit limit (optional)
4. Click **Simpan**
5. Customer is created with role `wholesale_customer`
6. Share login credentials with the customer
