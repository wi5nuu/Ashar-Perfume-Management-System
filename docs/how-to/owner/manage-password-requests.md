# Manage Wholesale Password Reset Requests

1. Navigate to **Wholesale Customers** page
2. Click the **Permintaan Password** tab
3. View pending requests in the **Pending** tab
4. Click **Setujui & Generate** to approve a request
5. The system generates a new password and shows it
6. Resolved requests appear in the **Terselesaikan** tab
