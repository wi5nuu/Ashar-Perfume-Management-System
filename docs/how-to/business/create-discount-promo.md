# Create a Discount or Promotion

1. Navigate to **Products** → **Promotions**
2. Click **Tambah Promo**
3. Configure the promotion:
   - Name & description
   - Discount type (percentage / fixed amount)
   - Discount value
   - Start & end date
   - Applicable products or categories
   - Minimum purchase (optional)
4. Click **Simpan**
5. The discount is applied automatically at checkout

## Notes

- Percentage discounts apply before tax
- Multiple promotions cannot be stacked per transaction
- Promotions can be toggled active/inactive without deleting
